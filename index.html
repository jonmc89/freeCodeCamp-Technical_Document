Introduction

HTML, or HyperText Markup Language, is the core language for creating and structuring web content. It uses tags and attributes to define elements like headings, paragraphs, links, and images. As the backbone of web development, HTML provides the framework for browsers to display content. Understanding HTML is essential for building or maintaining websites, as it underpins technologies like CSS and JavaScript.

  • Creating Web Pages: HTML is used to structure and format the content of web pages, including text, images, links, and multimedia elements.
  • Embedding Media: HTML allows the embedding of various media types such as videos, audio files, and interactive content, enhancing the user experience on websites.
HTML Boilerplate

The HTML boilerplate is a basic template that serves as the starting point for any HTML document. It includes essential elements such as the declaration, , , and tags. The boilerplate ensures that the document is properly structured and compatible with web standards, providing a solid foundation for further development. By using a boilerplate, developers can quickly set up a new project with all the necessary components in place.

When using an editor such as VS Code you can put in !! then press TAB this will automatically generate a useable HTML Boierplate.

  • First element in a boiler plate is HTML lang=en container.
  • Within the HTML tags a Head container with opening and closing tags.
  • Within the Head container the following will be added
    • One of more Meta elements.
    • One Title element.
    • One or more Link elements to link you CSS or external stylesheets.
  • The last thing to be added under the Head element will be a Body element with an opening and closing tag.
The Paragraph Element

The p element in HTML is used to define paragraphs of text. It is a block-level element, meaning it starts on a new line and takes up the full width available. Paragraphs are essential for organizing content into readable sections, making it easier for users to understand and navigate the text. The p element can contain text, inline elements, and other HTML tags, providing flexibility in formatting and styling.

The Image Element

The img element in HTML is used to embed images into a webpage. It is an inline element that requires the src attribute to specify the path to the image file, and the alt attribute to provide alternative text for accessibility. The img element enhances the visual appeal of a webpage and can be used to display logos, photos, illustrations, and other graphical content. Proper use of the alt attribute ensures that images are accessible to users with screen readers and improves SEO.

  • Within the img an src="" property is added, to which you need to add the URL of a web based image or the project file path for the locally stored image file.
  • The alt="" property is good practice to add with a description of what the image is incases of the iimage not loading.
The Anchor Element

The a element in HTML, also known as the anchor tag, is used to create hyperlinks that connect one webpage to another. It requires the href attribute to specify the destination URL. The anchor tag can enclose text, images, or other HTML elements, making them clickable. Hyperlinks are fundamental to web navigation, allowing users to move seamlessly between different pages and resources on the internet.

  • Within the a element you will need to add the href="" property, and then specify the object you with to navigate to.

Further Anchor Element Properties

  • Rel: The rel attribute in an anchor (a) element specifies the relationship between the current document and the linked document, enhancing security, privacy, and SEO.
  • Target: you can set this property to target="_blank" to open the link in a new tab.
